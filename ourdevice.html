SolUmbra's Features

Solar Array & Drawer System

To maximize energy harvest in a portable footprint, SolUmbra features a drawer-style retraction system that allows the solar panels to be securely stowed during transport.

Linear Actuator Lift Deployment

Powered by precision linear actuators, the system raises the solar array to clear ground-level obstructions and optimize sun exposure.

Smart Push-Assist Mobility

SolUmbra has an electric drivetrain featuring Push-Start Assist technology. By integrating a load cell into the handle, the device senses the user's manual force and provides proportional motor power to help "carry the load."

User-Centric Data Terminals

Engineering data is made accessible through a combination of an onboard OLED display for "at-a-glance" local status updates and a wireless interface for remote monitoring. This integrated system allows users to track battery health, system performance, and real-time environmental data from either the device itself or a connected external interface.
